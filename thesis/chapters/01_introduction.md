# 第1章 绪论

## 1.1 研究背景与意义

### 1.1.1 胶质母细胞瘤的临床特征与分型现状

胶质母细胞瘤（Glioblastoma，GBM）是成人最常见的原发性恶性脑肿瘤，约占所有恶性脑肿瘤的50% [1]（Stupp et al., 2005, N Engl J Med）。流行病学数据显示，GBM年发病率约为每10万人3至5例，且男性发病率显著高于女性 [2]（Ostrom et al., 2023, Neuro-Oncology）。尽管目前标准治疗方案——最大安全范围手术切除联合术后同步放化疗（Stupp方案）——已使患者中位总生存期（Overall Survival，OS）从不足12个月延长至约15个月，但GBM患者的5年生存率仍低于5% [1]（Stupp et al., 2005, N Engl J Med）。这一极差的预后状况，本质上源于GBM的高度异质性和浸润性生长模式——肿瘤细胞沿白质纤维束扩散，导致影像学上的肿瘤边界与实际浸润范围之间存在显著差异，使得根治性切除在多数情况下难以实现。

为应对这种异质性，建立能够区分不同预后层次的分型体系成为临床的核心需求。2021年，世界卫生组织（WHO）发布了第5版中枢神经系统肿瘤分类标准（CNS WHO 2021），完成了胶质瘤分类史上最根本的范式转变：首次将分子标志物纳入分型的核心依据 [3]（Louis et al., 2021, Neuro-Oncology）。根据新分类，"胶质母细胞瘤"这一诊断术语被限定为IDH野生型——既往所称的"IDH突变型胶质母细胞瘤"已被重新命名为"星形细胞瘤，IDH突变型，CNS WHO 4级" [4]（Berger & Mellinghoff, 2022, JAMA Oncol）。此外，即使组织学表现为低级别（WHO 2或3级）的弥漫性星形细胞瘤，若存在TERT启动子突变、EGFR扩增或染色体+7/-10共变异之一，亦应按照胶质母细胞瘤进行诊断和处理 [3]（Louis et al., 2021, Neuro-Oncology）。这一更新将分子特征置于与组织学形态同等甚至更优先的诊断地位，深刻改变了GBM的临床诊疗路径。

[图1-1插入位置：GBM分型体系与发展脉络——WHO组织学分类 → 2021分子整合分型 → 临床预后差异]

在分子预后标志物中，异柠檬酸脱氢酶（IDH）突变状态与O⁶-甲基鸟嘌呤-DNA甲基转移酶（MGMT）启动子甲基化状态被确认为最具临床指导价值的两个核心指标。Hegi等人针对Stupp临床试验开展的标志性相关研究表明，MGMT启动子甲基化患者接受替莫唑胺联合放疗的中位总生存期可达21.7个月，显著优于未甲基化患者的12.7个月（校正风险比HR=0.41，P<0.001）[5]（Hegi et al., 2005, N Engl J Med）。后续研究进一步揭示，IDH突变与MGMT甲基化之间存在协同预后效应——IDH突变且MGMT甲基化的患者预后最佳，而IDH野生型且MGMT未甲基化的患者预后最差，两组间的中位OS差异可达15个月以上 [6]（Molenaar et al., 2014, Neuro-Oncology）。

[图1-2插入位置：IDH突变型与野生型、MGMT甲基化与非甲基化患者的生存曲线对比示意图]

然而，上述分子分型体系在临床实践中面临根本性的实施困境。IDH突变与MGMT甲基化状态的检测均依赖于手术切除或立体定向活检获取的肿瘤组织样本 [7]（Seyhan, 2024, Int J Mol Sci）。脑活检操作存在一定的神经系统损伤风险，部分位于功能区的肿瘤因手术可及性受限而无法安全取材。更为突出的是，GBM在空间和时序维度上均表现出显著的分子异质性——同一肿瘤内部的不同区域可能呈现不同的突变谱，单次取材无法完整捕获这一异质性全貌 [7]（Seyhan, 2024, Int J Mol Sci）。此外，在治疗监测过程中，反复实施组织活检在实际临床操作中并不可行，致使临床医生缺乏实时、动态的分子信息来指导治疗调整。液体活检技术虽被寄予厚望，但受限于血脑屏障对循环肿瘤DNA向血液中释放的阻隔作用，其检测灵敏度在GBM中远未达到临床实用水平 [7]（Seyhan, 2024, Int J Mol Sci）。

上述困境引出了一个具有重要临床价值的研究方向：MRI作为GBM诊疗中不可或缺的无创影像工具，是否能够在一定程度上弥补分子检测的上述不足，提供与预后相关的补充信息？多项研究为此提供了令人鼓舞的证据。Rathore等人的研究表明，基于MRI和病理影像特征的影像表型模型在预测总生存期方面显著优于仅使用年龄、性别和IDH突变状态的基础模型，多组学整合模型的C-index达到0.87 [8]（Rathore et al., 2023, Comput Methods Programs Biomed）。进一步地，Guo等人通过对571例IDH野生型GBM的多参数MRI影像组学数据进行无监督聚类，发现了三个具有显著生存差异的影像亚型（HR=1.64，95% CI 1.17-2.31），证实了影像特征在无监督条件下发现预后相关亚型的可行性 [9]（Guo et al., 2024, Sci Rep）。此前的一项早期研究同样报道了基于267个MRI影像组学特征的无监督聚类可识别出三种具有不同分子特征和生存结局的GBM亚型（p<0.001）[10]（Rathore et al., 2018, Sci Rep）。

上述研究共同指向一个核心科学问题：如果多模态MRI图像本身已经包含了与肿瘤生物学行为和患者预后密切相关的异质性信息，那么是否能够在完全不依赖分子检测的前提下，通过深度无监督聚类方法从多模态MRI中自动发现具有预后差异的潜在影像亚型？若能实现，这将为GBM患者的无创预后分层提供一种新的可能性，使影像学从单纯的"定位诊断工具"向"预后信息载体"的方向演进。这正是本文拟探索的核心问题。

### 1.1.2 多模态MRI在胶质瘤评估中的应用

磁共振成像（MRI）凭借其优异的软组织对比度与无电离辐射特性，已成为GBM临床诊疗中不可或缺的影像学手段——贯穿术前诊断、手术规划、疗效评估及复发监测的全流程。为全面获取肿瘤信息，临床实践中形成了标准化的多模态MRI扫描方案。Ellingson等人代表国际脑肿瘤影像标准化委员会发布的共识建议，明确了脑肿瘤临床试验的最低序列要求：参数匹配的三维T1加权成像、钆对比剂增强后T1加权成像（T1c）、轴位二维T2加权成像以及轴位二维T2液体衰减反转恢复序列（FLAIR）[11]（Ellingson et al., 2015, Neuro-Oncology）。这四种序列通过不同的组织对比机制，各自突出GBM的不同病理特征，共同构成描述肿瘤全貌的影像学基础。

四种模态本质上捕获的是肿瘤不同维度的生物学信息。T1c模态在静脉注射钆对比剂后采集，利用GBM活跃增殖区域血脑屏障破坏导致的对比剂外渗效应，直接标记肿瘤的活性核心——即血脑屏障破坏最严重、肿瘤细胞增殖最活跃的区域[12]（Bakas et al., 2017, Scientific Data）。FLAIR模态通过特殊的反转脉冲抑制脑脊液信号，使位于脑室旁及皮层表面的肿瘤边界和瘤周异常信号区得以清晰呈现。Scola等人的综述指出，FLAIR与T2加权图像上围绕肿瘤的瘤周高信号区域（non-enhancing peritumoral area）是GBM影像学评估中的关键结构——该区域不仅包含血管源性水肿成分，更可能隐藏着沿白质纤维束浸润的肿瘤细胞，其影像特征与患者预后和治疗反应密切相关[13]（Scola et al., 2023, Cancers）。T1平扫提供的是肿瘤相对于正常脑组织的解剖学参考基线，用以排除T1固有高信号（如出血或钙化）对增强评估的干扰。T2加权图像则对组织整体水含量变化敏感，能够较全面地反映肿瘤占位效应和水肿的空间分布范围。四者的关注点各有侧重：T1c锁定活性肿瘤核心，FLAIR勾勒肿瘤向周围脑组织浸润的最远边界，T2揭示水肿的整体空间分布，而T1为上述各模态的对比信号提供解剖参考基准。

[图1-3插入位置：同一患者同一横断面下的FLAIR / T1 / T1c / T2四模态MRI影像对比图]

上述四种模态之间的信息关系是互补而非冗余的。若仅依赖T1c图像进行评估，临床关注范围将被局限于血脑屏障已遭破坏的活性肿瘤区域，而研究表明，GBM的浸润范围往往超出T1c所示的强化边界——FLAIR高信号区域内的非强化部分同样含有肿瘤细胞浸润，且该区域的扩大常常先于T1c强化灶的显现[13]（Scola et al., 2023, Cancers）。反之，FLAIR高信号区域虽敏感度较高，但其特异性存在固有局限：肿瘤浸润性水肿与单纯血管源性水肿在常规FLAIR图像上信号特征相近，仅凭单一序列难以做出可靠区分。T1平扫和T2加权序列同样各自仅能提供单一维度的信息——前者缺乏对肿瘤活性的直接指示能力，后者无法分辨不同类型的水肿成分。归根结底，单一MRI模态所呈现的仅为GBM复杂异质性图景中的一个侧面；任何试图基于单一模态完整刻画肿瘤特征的方案，都将面临固有的信息缺失。

因此，如何有效整合四种模态所蕴含的互补信息，成为基于多模态MRI进行GBM无监督聚类分析必须首先解决的方法学问题。最直接的融合策略——将四种模态在体素或特征层面进行简单拼接——虽然操作便捷，但忽略了不同模态间潜在的语义交互关系：T1c上的强化区域与FLAIR上的高信号区域在空间上存在对应、嵌套或偏移等关联模式，这些跨模态的空间关系可能携带着与肿瘤生物学行为相关的重要信息。简单拼接无法捕获此类跨模态交互；将各模态交由独立分支编码再在决策端汇总的晚期融合策略，同样损失了模态间在中间表征层面的交互机会。这一方法学需求自然引出了本文需解决的首要技术问题——在无监督聚类框架下，如何设计有效的多模态融合机制以实现多模态MRI信息的整合利用。该问题将在1.1.3节中得到进一步的具体化。

### 1.1.3 选题意义

前文分别论证了两个递进的观点：1.1.1节指出，基于多模态MRI无创发现具有预后差异的影像亚型，是当前GBM临床诊疗中一个具有明确需求但尚未解决的科学问题；1.1.2节进一步阐明，多模态MRI各序列所捕获的肿瘤生物学信息是互补的，有效整合这些信息是开展影像亚型分析的前提。然而，从"这一目标值得追求"到"这一目标可以达成"之间，存在着三个环环相扣的方法学障碍。本节的目的是将这三个障碍明确表述为具体的研究问题——它们共同构成了本文的技术动机与选题依据。

第一个问题是如何有效融合四种MRI模态之间的异构信息。1.1.2节已阐明，T1、T1c、T2和FLAIR四种序列的成像物理机制不同，各自突显的是GBM不同病理特征的侧面。这种信息差异不仅是"各有所长"，更本质性地表现为信息异构——不同模态的信号值分布范围、空间分辨率特征和组织对比产生方式均存在系统性差异。当这些模态被作为聚类模型的输入时，异构性会引出一个直接的方法学困境。最为通行的早期融合策略——在体素或特征层面将四种模态直接拼接为多通道张量——隐含地将异构信息视为同质的多模态数据，导致信号强度较高的模态（如FLAIR）在特征提取初期即可能占据主导地位，弱化其他模态的贡献。晚期融合策略——各模态先经由独立分支编码再在决策端整合——虽然保护了各模态特征在初期不被稀释，但代价是完全丧失了模态间在特征学习过程中的交互机会。而在GBM多模态影像中，模态之间的空间关系本身就可能是具有预后意义的影像标志：例如T1c强化核心相对于FLAIR高信号区域的位置偏移、大小比例和嵌套形态等跨模态空间关系模式，均无法在"各自为政"的晚期融合框架下被有效捕获。因此，在无监督聚类框架下设计能够兼顾模态特异性与跨模态交互的融合机制，构成本文需要解决的第一个核心问题。

第二个问题是深度聚类中如何维持稳定且具有区分度的特征表示。有监督分类任务拥有明确的标签信号作为优化目标——梯度沿"减小预测与标签差异"的方向传播，特征空间在训练中自然倾向于形成类间可分的结构。而无监督聚类缺乏这种外部锚定信号：特征学习与聚类目标必须同步从数据本身中被挖掘和构建。这带来了被已有研究明确指出的"特征塌缩"风险——在未施加足够约束的情况下，深度网络倾向于将所有输入样本映射到特征空间中一个极小的邻域，使得嵌入向量丧失区分不同样本的能力。Bardes等人在自监督表示学习的研究中已证实，若不施加显式的方差与协方差正则化，联合嵌入架构极易输出退化的恒定特征表示，导致下游聚类任务完全失效[14]（Bardes et al., 2022, ICLR）。在GBM多模态影像聚类这一特定场景中，上述问题因数据特性而进一步加剧：3D MRI体素数据本身维度极高但样本量相对有限（通常仅数十至数百例），高维小样本的学习环境使网络更倾向于通过过度压缩信息维度来降低优化目标，而非保留对聚类有价值的结构差异。因此，如何在无监督条件下设计有效的表示学习正则化策略，在维持特征空间区分度的同时避免塌缩，构成本文需要解决的第二个核心问题。

第三个问题是聚类簇数的自动确定与聚类塌缩的协同控制。传统聚类方法——无论是K-Means还是高斯混合模型——均要求用户在分析前预先指定簇数K。然而在GBM影像亚型发现的任务中，最优亚型数目本身正是需要回答的首要科学问题：预设K=2、3还是4，将直接影响后续的生存分析结论和生物学解释。已有影像组学聚类研究中对此的处理方式多为事后选择——即分别以多个K值运行聚类，再通过内部评价指标或外部生存差异显著性来筛选"最优"K[9]（Guo et al., 2024, Sci Rep）。这种做法存在两个固有局限：其一，K的选择过程本身已包含了主观判断，削弱了无监督聚类应有的探索属性；其二，事后选择本质上是"先以错误K聚类、再从中挑选相对最好的结果"，无法回避初始K设置不当对簇内结构造成的扭曲。更为隐蔽且在实践中更具破坏性的是"聚类塌缩"现象：即便设定了名义上的K个簇，模型也可能将绝大多数样本集中分配到极少数簇甚至单个簇中——名义簇数与有效簇数严重偏离。Ronen等人在DeepDPM的工作中明确指出，这种塌缩是无标签深度聚类中最常见且后果最严重的失效模式之一，其根源在于聚类损失函数在无外部约束时存在使簇分配熵极小化的退化方向[15]（Ronen et al., 2022, CVPR）。"自动确定K"与"防止聚类塌缩"实际上构成了一对需要协同解决的矛盾：若分配机制缺乏足够约束，网络可能通过将所有样本归入极少数簇这一捷径来最小化聚类损失；若通过强制均匀分配来抵抗塌缩，又可能扭曲数据本身可能具有的不均匀簇分布结构。如何在两者之间建立有效的平衡，构成本文需要解决的第三个核心问题。

上述三个问题并非互不相关的技术孤岛，而是构成了一条串行依赖的完整方法学链路。多模态融合质量决定了聚类模块输入特征的信息完备性；表示学习的稳定性决定了特征空间是否保有足够的结构以支撑有意义的聚类划分；自动簇数确定与塌缩控制则直接决定了聚类结果在临床解读中是否具备可解释性和可应用性。三者中的任何一环存在薄弱之处，都将经由链路传导而影响最终的影像亚型发现质量。本文的研究目标正是围绕这三个耦合问题，构建一个完整的深度无监督聚类框架——在统一的端到端训练中同时解决多模态异构融合、稳定表示学习与自适应簇数推断三项任务。1.2节将对上述各关键技术方向的已有研究进行系统梳理与批判性分析，1.3节则进一步明确本文的具体研究内容与技术路线。

## 1.2 国内外研究现状

### 1.2.1 基于深度学习的医学图像聚类方法

传统聚类方法——如K-Means、高斯混合模型和谱聚类——的聚类质量高度依赖于输入特征的表征能力。在医学图像分析中，这些特征通常由人工定义的影像组学描述子构成（如一阶灰度统计量、纹理特征和形状描述子），其特征空间的组织方式受限于研究者的先验知识。深度聚类方法的出现为突破这一瓶颈提供了新的思路：通过将深度神经网络的表示学习能力与聚类目标函数进行联合优化，特征提取与簇分配可以在端到端的统一框架中相互促进，使学习到的特征表示直接服务于聚类任务的需求。Lu等人在综述中依据所引入先验知识的类型，将深度聚类方法归纳为六种范式——包括基于结构先验的自编码器重构约束和基于分布先验的KL散度匹配——并指出深度聚类的核心挑战在于：在缺乏外部标签信号提供明确优化方向的前提下，如何引入足够强的归纳偏置来引导特征空间向有利于聚类分离的方向组织[19]（Lu et al., 2024, Vicinagearth）。

在深度聚类的技术演进脉络中，Xie等人提出的DEC（Deep Embedded Clustering）具有奠基性意义。DEC首次将特征学习与聚类分配统一为端到端的优化框架：首先通过堆叠去噪自编码器对数据进行逐层预训练以获取初始特征表示；随后丢弃解码器，在嵌入空间中以K-Means初始化聚类中心；最终通过迭代最小化软分配分布q与辅助目标分布p之间的KL散度，同步更新编码器参数与聚类中心位置[16]（Xie et al., 2016, ICML）。其中辅助分布p通过对当前软分配q进行平方归一化构造，使高置信度分配获得更高的目标权重，形成一种"自训练"式的优化闭环。这一"软分配+辅助目标"机制为后续大量深度聚类方法确立了基本的优化范式。

然而，Guo等人在IDEC（Improved Deep Embedded Clustering）中对DEC提出了一项关键的修正。IDEC指出，DEC丢弃解码器的操作在提升聚类效率的同时付出了被忽略的代价——嵌入空间丧失了维持原始数据局部邻域结构的能力。IDEC在DEC的KL聚类损失之上重新引入了自编码器的重构损失，以约束编码器在向聚类目标优化的同时保留足够的局部结构信息来支撑解码器的重建[17]（Guo et al., 2017, IJCAI）。该设计揭示了一个在深度聚类中普遍存在但易被低估的权衡关系：聚类目标天然倾向于将特征空间向离散的簇中心压缩，而结构保持目标则需要维护数据的连续流形结构——二者的张力直接决定了聚类结果能否忠实反映数据本身的分布特征。在GBM多模态MRI聚类场景中，这一权衡尤为关键：肿瘤的影像表型在群体中更接近于连续谱分布而非截然分离的离散簇，过强的聚类驱动力可能导致对连续表型渐变的人为割裂。

[图1-4插入位置：典型深度聚类方法流程对比图（DEC / IDEC / DeepCluster）]

在DEC与IDEC所代表的"嵌入空间内聚类"路线之外，Caron等人提出的DeepCluster开辟了另一条技术路径——伪标签迭代机制。DeepCluster并不直接在嵌入空间内优化簇分配，而是交替执行两个步骤：使用K-Means对当前CNN提取的全数据集特征进行聚类以生成伪标签，随后将这些伪标签视为监督信号来训练CNN进行分类[18]（Caron et al., 2018, ECCV）。这一迭代范式表明，在无监督条件下，"聚类→伪标签→特征学习→重新聚类"的循环不仅可行，而且伪标签的质量会随着特征学习轮次的推进而逐步改善。然而，DeepCluster的设计立足于大规模自然图像数据集（ImageNet，百万级样本），其每轮均需遍历全数据集执行K-Means的计算模式在3D医学图像的样本规模下效率显著降低，且其依赖的固定K预设同样无法直接适配GBM亚型数目未知的研究场景。

综上，上述深度聚类方法为无监督特征学习与聚类联合优化提供了重要的方法论要素——DEC的软分配自训练机制、IDEC的结构保持正则化、DeepCluster的伪标签迭代思路。但将其直接迁移至GBM多模态MRI聚类场景时，面临三个层面的系统性局限：其一，数据形态层面，上述方法均基于2D自然图像设计，骨干网络无法直接捕获3D MRI体素数据中的空间结构特征；其二，模态维度层面，方法假设输入为单一模态，不具备处理四种MRI异构信息的能力，多模态融合模块的设计需从零嵌入聚类框架；其三，稳定性层面，无监督聚类结果对初始化、超参数和数据采样顺序高度敏感，而医学数据分析对结果的稳健性和可复现性有远高于计算机视觉基准实验的要求。因此，现有深度聚类方法虽提供了重要的技术基础，但尚不足以直接应对GBM多模态MRI无监督影像亚型发现的完整需求。

### 1.2.2 多视图医学图像融合方法

多模态MRI数据天然具备多视图属性——T1、T1c、T2和FLAIR四种序列从不同的物理对比机制出发，各自描述GBM肿瘤的不同生物学侧面。如何将这些异构的多视图信息整合为统一的特征表示，是多模态医学图像分析中的基础性方法学问题。依据融合操作在模型架构中所处的深度位置，现有融合策略可划分为三种基本范式。早期融合在模型的输入层将多模态体素直接拼接为多通道张量，由后续网络自主从拼接后的数据中提取融合特征。晚期融合令各模态分别经过独立的编码分支完成特征提取，仅在最终的决策端对各模态的特征或预测进行整合。中期融合则在特征编码的中间层级引入跨模态的信息交互，力求在模态特异性保护与多模态信息互补之间取得折中平衡。三种范式的取舍，本质上反映了对不同模态间"独立性"与"交互性"之间平衡点的不同选择。

早期融合虽操作最为简捷，但在多模态MRI场景中面临一个根本性困难：不同模态的信号值分布范围存在系统性差异。FLAIR序列在肿瘤和水肿区域呈现高信号，T1c的增强信号局限于血脑屏障破坏区，T1平扫的信号动态范围则相对较窄。体素级直接拼接将这些数值差异显著的不同模态视为同质的输入通道，灰度值较高的模态可能在中底层卷积的特征响应中占据主导地位，使得信号相对较弱但同样具有诊断价值的模态信息在特征提取的早期阶段即被稀释弱化。晚期融合则走向了另一个极端——各模态通过完全独立的编码分支进行特征提取，虽保护了各模态特征不被跨模态数值差异所干扰，但以完全牺牲模态间的中间表征交互为代价。已在1.1.2节中讨论过的T1c强化区与FLAIR高信号区之间的空间嵌套与偏移关系等跨模态关联模式，在晚期融合框架下完全无法被模型所捕获和利用。

为在上述两种极端之间寻求更优的折中，基于注意力机制的中期融合方法近年来逐渐成为多模态医学图像分析中的主流技术方向。其核心设计思想在于：通过计算不同模态特征图之间的相似度矩阵（注意力权重），使每一模态能够根据当前语义上下文，动态地、有选择性地关注其他模态中的相关空间区域或特征通道，从而实现内容感知的软性信息融合。Lin等人在脑肿瘤多模态MRI分割任务中提出的模态关联交叉注意力（Modality-Correlated Cross-Attention，MCCA）是这一方向的代表性工作——该方法依据MRI模态的物理成像原理将四种序列划分为两个语义组（T1/T1c对应解剖与增强对比，T2/FLAIR对应水肿与水含量信息），并在两组之间施加结构化的交叉注意力以实现受控的跨模态信息交换，在BraTS基准上取得了领先的分割精度[20]（Lin et al., 2023, IEEE TMI）。

[表1-1插入位置：多视图融合方法分类对比表（早期融合 / 晚期融合 / 中期融合 / 注意力融合）]

然而，上述以跨模态注意力（Cross-Attention）为代表的先进融合机制共享一个关键的隐性前提：融合模块的注意力权重依赖于监督任务损失（如分割的逐像素交叉熵和Dice损失）提供的明确梯度信号来进行学习。在无监督聚类场景中，监督梯度被替换为由聚类软分配导出的KL散度损失，其对注意力权重的引导强度远弱于像素级的密集监督信号。当模态间注意力缺乏有效学习信号时，可能退化为近似均匀的信息混合，丧失对模态进行差异化选择和筛选的能力。此外，从已系统检索的文献来看，多模态MRI融合研究目前高度集中于有监督的分割与分类任务，在无监督聚类框架下探索跨模态特征融合机制的工作尚属空白——这也意味着，本文在借鉴有监督融合中注意力机制等核心设计理念的同时，需要独立探索这些机制在无监督条件下的有效实现与适配方式。

### 1.2.3 自动簇数确定技术

K-Means、谱聚类和高斯混合模型等最常使用的聚类方法，均要求用户在分析启动之前预先指定目标簇数K。在聚类作为探索性数据分析工具使用的场景中，K的取值可通过肘部法则、轮廓系数或Gap Statistic等启发式指标辅助确定；然而在GBM影像亚型发现的特定任务中，最优亚型数目本身即是研究者试图回答的首要科学问题——预设K=2意味着一个先验假设（GBM存在两种影像亚型），K=3则对应另一种假设，而不同的K选择将直接导致不同的生存分析结论和生物学解释路径。已有研究中对此的通行处理方式——分别以多个候选K值运行聚类，再通过内部有效性指标或外部预后差异显著性来事后筛选"最优"K[9]（Guo et al., 2024, Sci Rep）——虽然在操作层面解决了"选K"的工程需求，但存在方法学层面的深层局限：事后选择已将研究者对"何种亚型划分是好的"这一价值判断嵌入了分析流程，削弱了无监督聚类作为数据驱动探索工具的方法论独立性。

Dirichlet Process（DP）为非参数贝叶斯统计提供了绕过预设K的形式化框架。其核心数学直觉可由Sethuraman提出的Stick-Breaking构造过程直观阐释：将一根单位长度的"棍子"逐次折断——首次折断的比例β₁确定第一个簇的混合权重π₁=β₁，第二次从剩余长度（1-β₁）中折断比例β₂，得到π₂=β₂(1-β₁)，由此递推生成一个随簇索引k增大而单调递减的权重序列[21]（Sethuraman, 1994, Statistica Sinica）。这一构造天然编码了一个合理的先验信念——数据中占主导地位的少数簇应获得较大的混合权重，而新增簇的重要性随簇数增多而逐步衰减。浓度参数α作为先验超参数控制权重衰减速率，使模型可在不事先设定簇数上限的条件下，从数据中自适应地学习实际有效的簇数目。

将DP引入深度神经网络框架面临的核心技术障碍在于离散采样操作的不可微性——Stick-Breaking过程中每一步的折断比例β_k需从Beta分布中采样，这一操作不直接支持梯度反向传播。Nalisnick和Smyth提出的Stick-Breaking VAE首次在神经网络的变分推断框架中嵌入该过程，利用重参数化技术实现Beta采样的可微近似，展示了在深度生成模型中表达无限混合表示的可行性[22]（Nalisnick & Smyth, 2017, ICLR）。同期，Jang等人提出的Gumbel-Softmax分布为离散随机变量的连续松弛提供了更为通用的技术方案——通过向离散类别的log概率中添加Gumbel噪声，再经温度参数τ控制的softmax变换，使得原本不可微的离散采样在τ>0时具有连续且可微的输出[23]（Jang et al., 2017, ICLR）。上述两项技术进展为将非参数贝叶斯簇数推断嵌入端到端深度聚类框架奠定了实现基础。

[图1-5插入位置：非参数聚类技术发展时间线（Sethuraman 1994 → Stick-Breaking VAE 2017 → Gumbel-Softmax 2017 → DeepDPM 2022）]

Ronen等人提出的DeepDPM是目前最具代表性的端到端自动K深度聚类方法。该方法以DP混合模型为理论基础，通过Split/Merge蒙特卡洛采样在训练过程中动态调整聚类簇数，在ImageNet等大规模基准数据集上验证了无需预设K的深度聚类的可行性[15]（Ronen et al., 2022, CVPR）。然而，DeepDPM的设计也揭示了自动K方法在深度聚类实际应用中所面临的若干关键挑战：MCMC采样过程引入了额外的计算开销和收敛诊断难度；Split/Merge操作依赖预设的接受阈值参数，而阈值的选择对最终簇数具有实质影响。更为核心的是，自动K推断机制与聚类塌缩防控之间尚未建立充分的耦合约束——即便理论上允许K根据数据自适应地调整，若聚类损失函数中存在使簇分配熵趋于极小化的退化方向，模型可能在训练过程中将所有样本推入极少数簇，导致有效簇数塌缩至1或2。"能够自动确定K"并不等同于"能够在训练中稳定地维持合理的有效K值"——前者解决的是簇数的选择机制问题，后者则要求同时在训练动态中持续抵抗聚类的塌缩倾向。两者的协同解决，是自动K方法从计算机视觉基准验证走向医学影像聚类这一高可靠性要求应用场景之前必须跨越的关键障碍。

### 1.2.4 研究现状总结与本文切入点

综合1.2.1至1.2.3节对三个关键技术方向已有研究的系统梳理，可以提炼出一条贯穿各方向的共性判断：现有方法在"无监督聚类"这一约束条件下，对"多模态3D MRI数据"这一特定应用场景的适配存在系统性的不足。该不足并非孤立地体现在某一方向的技术指标上，而是在多模态融合、表示学习稳定性和簇数推断三个层面的交汇地带形成了方法学的空白区域。

现有研究与本文目标场景之间存在三条递进的关键差距。第一，在多模态融合层面，跨模态注意力等先进融合机制已在有监督分割任务中证明了其对多模态交互建模的有效性，但其注意力权重的学习高度依赖密集监督信号的引导，尚缺乏在无监督聚类条件下实现有意义跨模态特征交互的验证方案。第二，在表示学习层面，深度聚类所依赖的特征空间正则化策略——如IDEC中以解码器重构约束维持局部结构的做法——在3D MRI体素数据的高维低样本条件下能否有效移植，尚未得到充分的检验；医学数据本身的小样本特性进一步压缩了可用正则化策略的试错空间。第三，在自动簇数推断层面，可微非参数贝叶斯方法为实现端到端的自动K学习展示了技术可行性，但其与聚类塌缩控制机制之间的协同设计尚未被充分探索——两组机制分别回应"选K"和"稳聚类"的需求，但在一个统一的训练框架中同时保障两者有效性的方案尚属缺失。

上述三条差距共同界定了本文研究工作的切入方向。针对多模态融合的不足，本文设计多模态编码器以在无监督条件下实现模态间语义交互的融合机制。针对表示学习稳定性的不足，本文引入并调整方差-协方差正则化策略以在无标签约束下维持特征空间的区分度。针对自动K与塌缩控制的协同需求，本文构建基于可微Stick-Breaking的簇数推断模块，并将其嵌入带有防塌缩约束的聚类优化框架之中。三个模块并非相互独立的功能组件，而是以聚类质量为共同优化目标，在一个统一的端到端训练流程中协同运作。1.3节将在此基础上进一步明确本文的研究目标、主要研究内容与技术路线。

## 1.3 研究目标与内容

### 1.3.1 研究目标

综合1.1节对研究背景的分析与1.2节对研究现状的梳理，本文的核心研究目标是：针对GBM多模态MRI数据的异构性、高维性与无标签特性，构建一个完整的深度无监督聚类框架，在不依赖分子检测和预设簇数的前提下，从多模态MRI中自动发现具有预后差异的潜在影像亚型。围绕这一总目标，本文分解出以下三个具体研究目标。

第一，建立面向多模态MRI的统一特征表达。T1、T1c、T2和FLAIR四种模态在成像物理机制和信息侧重上的差异（详见1.1.2节），使得直接拼接或多分支独立编码均无法有效整合其互补信息（详见1.2.2节）。本文的研究目标是设计一种能够在保留各模态特异性信息的同时、捕获跨模态语义交互的特征融合机制，为下游聚类任务提供信息完备且紧凑的肿瘤影像表示。

第二，实现稳定的无监督深度聚类。与有监督分类不同，无监督聚类缺乏外部标签信号提供的明确优化方向，深度网络在聚类损失的驱动下容易出现特征塌缩——将所有样本映射到特征空间中的极小邻域，导致嵌入向量丧失区分不同样本的能力（详见1.2.1节）。本文的研究目标是在不依赖任何标签信息的前提下，通过设计合适的表示学习正则化策略与聚类优化机制，维持特征空间的类内紧凑性与类间可分性，确保聚类结果具有统计意义上的稳定性和可复现性。

第三，实现聚类簇数的自动确定与聚类塌缩的有效防控。传统聚类方法对预设K的依赖与GBM影像亚型数目未知这一事实之间存在根本矛盾；同时，即便在指定K的条件下，无监督深度聚类仍频繁出现簇分配塌缩——绝大多数样本被分配给极少数簇（详见1.2.3节）。本文的研究目标是将非参数贝叶斯中的自适应簇数推断思想引入深度聚类框架，使模型能够在端到端训练中自动学习合理的簇数目，并同步建立防塌缩约束以维持有效簇数的稳定。

上述三个目标并非彼此独立，而是围绕"从多模态MRI中进行高质量的影像亚型无监督发现"这一总目标形成了递进支撑关系：统一特征表达为聚类提供信息基础，稳定表示学习保障聚类可用的特征空间质量，自动簇数确定与塌缩防控则直接决定聚类结果的有效性和可解释性。

### 1.3.2 主要研究内容

为实现上述研究目标，本文的研究内容按照"数据预处理—特征编码与融合—聚类机制设计—训练策略构建—实验验证"的逻辑链路组织，具体包括以下四个部分。

第一，多模态MRI数据预处理与特征编码融合。本研究的输入数据为GBM患者的四种标准MRI序列（FLAIR、T1、T1c、T2），所有图像经配准至统一坐标空间、颅骨剥离和灰度归一化后，输入深度网络进行特征提取。特征编码部分的设计遵循"先模态内提取、后模态间融合"的两阶段原则：首先使用3D卷积神经网络对各模态分别提取体素级的局部空间特征，随后通过跨模态特征交互模块在四个模态之间建立信息通路，使各模态的特征表示能够根据互补信息进行动态调整。最终输出的是一个融合了四种模态互补信息的统一肿瘤特征表示，作为后续聚类模块的输入。

第二，基于非参数思想的自动聚类数确定机制。针对传统聚类方法必须预设簇数K的固有限制，本文引入Dirichlet Process的Stick-Breaking构造思想，在聚类模块中维护一个可学习的簇权重分布。在训练过程中，模型通过温度参数退火调度逐步收缩权重趋近于零的无效簇，从而在不需要人工指定K的条件下，自动从数据中推断出合理的有效簇数目。这一机制使得聚类结果对K取值的敏感性不再构成影响研究结论的外部干扰因素。

第三，稳定聚类训练策略与反塌缩机制。为应对深度聚类中普遍存在的特征塌缩与簇分配塌缩问题，本文从三个层面构建防御机制。在表示学习层面，引入并适应性调整方差-协方差正则化策略，在不依赖标签信号的条件下约束特征空间维持足够的维度活跃度和样本区分能力。在聚类优化层面，采用两阶段期望最大化训练策略——第一阶段冻结编码器参数，专注于聚类中心的稳定更新；第二阶段冻结聚类参数，专注于编码器特征的聚类适配——以交替优化的方式避免聚类信号与表示学习信号之间过早相互锁定导致的训练不稳定。在簇分配层面，通过熵约束和温度调度控制簇分配的锐度与均匀性之间的平衡，防止模型经由"将所有样本归入单一簇"的捷径来最小化聚类损失。

第四，聚类结果的实验评估与临床预后验证。本文对所提出的方法进行多维度的实验评估。在聚类质量层面，与K-Means、高斯混合模型、DEC和IDEC等代表性基线方法进行对比，使用轮廓系数（Silhouette Score）、Davies-Bouldin指数（DBI）和Calinski-Harabasz指数（CHI）等内部有效性指标进行定量评价。在临床预后验证层面，对聚类所发现的影像亚型进行Kaplan-Meier生存分析、Log-rank检验和多变量Cox比例风险回归，以评估不同亚型之间是否存在显著且独立于已知临床因素的生存差异。此外，通过对模型在不同训练阶段的簇数动态变化和簇分配分布演变进行追踪分析，对自动K机制和反塌缩策略的实际效果进行过程性诊断。

## 1.4 论文结构安排

本文共分为六章，各章内容组织如下。

第1章为绪论。本章首先从GBM的临床特征与分子分型现状出发，阐述了组织活检在分子检测中面临的侵入性局限，引出"基于多模态MRI无创发现预后相关影像亚型"这一核心研究问题；进而分析了四种标准MRI序列在GBM评估中的互补作用与技术融合需求；在此基础上，凝练出多模态异构融合、稳定表示学习和自动簇数确定三项关键方法学问题，明确了本文的选题依据。随后，对深度聚类、多模态融合和自动簇数确定三个方向的国内外研究现状进行了系统梳理与批判性分析。最后，阐明了本文的研究目标、主要研究内容与整体技术路线。

第2章为相关技术与理论基础。本章以本文方法链路的构建顺序为线索，依次介绍所涉及的基础技术：多模态医学图像融合的基本策略与从卷积到时序建模的特征提取方法、深度聚类从DEC到自监督正则化的核心范式演进、Dirichlet Process非参数贝叶斯框架及其可微松弛技术、以及生存分析的三重临床验证方法论。本章的目的是为第4章的方法设计提供必要的技术背景与理论依据。

第3章为数据集与预处理。本章系统介绍实验数据的来源、纳入与排除标准，详细描述四种MRI模态的成像特性与互补关系，并依次阐述MNI空间配准、体素重采样、Z-score强度归一化和模态缺失处理四项预处理操作的技术细节与设计考量。本章为后续的特征编码与聚类分析提供统一的数据输入格式。

第4章为方法设计。本章详细阐述本文所提出的基于多模态MRI的深度无监督聚类框架，依次展开四个核心模块的设计：多模态特征编码与Transformer跨模态融合、基于Gumbel-Sigmoid Stick-Breaking的自动聚类数确定、融合分布级聚类约束与两阶段期望最大化优化的聚类学习框架、以及包含协方差降权VICReg正则化与课程式熵约束在内的多层反塌缩训练策略。各模块的设计逻辑与技术细节均在对应小节中逐层展开，并阐明模块之间的数据流依赖与联合优化关系。

第5章为实验验证与结果分析。本章对所提出方法进行系统性的实验评估，包括聚类质量定量比较、临床预后验证、消融实验以及训练过程诊断分析。在聚类质量层面，与K-Means、高斯混合模型、DEC和IDEC等代表性基线方法进行对比；在临床预后层面，通过Kaplan-Meier生存曲线、Log-rank检验和Cox多变量回归检验聚类亚型的预后差异显著性。

第6章为总结与展望。本章对全文的研究工作进行回顾性总结，归纳本文的主要贡献与当前方法存在的局限性，并对未来在数据扩展、多中心外部验证、多组学数据整合以及半监督聚类框架等方向上的改进空间进行展望。
